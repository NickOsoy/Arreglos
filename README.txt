README:
En la Clase EjercicioLinkList contiene tres métodos estáticos que realizan las operaciones de las listas enlazadas:

Métodos
eliminarDuplicados: con este metodo se elimina los valores duplicados de una lista enlazada de enteros, dejando solo la primera aparición de cada número.

invertirLinkedList:este metod invierte el orden de los elementos de una lista enlazada de cadenas.

intercalarListas:este metodo Intercala dos listas enlazadas ordenadas de enteros en una nueva lista.


Clase EjercicioLinkListTest
La clase EjercicioLinkListTest contiene pruebas unitarias para verificar el correcto funcionamiento de los métodos de EjercicioLinkList en el cual se implemento Log4j para registrar los resultados de las pruebas.

Pruebas

testEliminarDuplicados: Verifica que el método eliminarDuplicados elimine correctamente los valores duplicados de una lista.

testInvertirLinkedList: Verifica que el método invertirLinkedList invierta correctamente el orden de los elementos de una lista.

testIntercalarListas: Verifica que el método intercalarListas intercale correctamente dos listas ordenadas.

testIntercalarListasForzarFallo: esta prueba esta diseñada para fallar, verificando que el método intercalarListas no devuelva una lista incorrecta.
